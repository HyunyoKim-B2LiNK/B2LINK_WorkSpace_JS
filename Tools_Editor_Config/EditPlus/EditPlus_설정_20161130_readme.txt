@. ctrl + Q : Edit Plus 종료

@. ctrl + N : 새문서 ( JavaScript )

@. ctrl + S : 현재 문서 저장
@. ctrl + W : 현재 문서 닫기
@. ctrl + shift + R : 현재 문서 다시 로드
@. F2 : 현재 문서 이름 바꾸기
@. ctrl + I : 현재 문서 인코딩 변경

@. ctrl + L : 줄번호 찾아가기
@. ctrl + D : 선택된 줄 전체 지움
@. ctrl + alt + Down : 줄 복제

@. ctrl + F : 찾기
@. ctrl + R : 바꾸기
@. ctrl + H : 여러 파일 찾기
@. ctrl + K : 다음 단어 찾기
@. ctrl + [ : 괄호 찾기

@. ctrl + shift + D : 선택 텍스트 Base64 디코딩
@. ctrl + shift + I : 선택 텍스트 Base64 인코딩
@. ctrl + shift + K : 이전 단어 찾기
@. ctrl + shift + F : 현재 파일 디렉토리에서 찾기 ( 하위폴더 포함 )
@. ctrl + shift + S : 새 이름으로 현재 문서 저장
@. ctrl + shift + X : 대문자로
@. ctrl + shift + Y : 소문자로
@. ctrl + shift + Z : 대소문자 뒤바꿈

@. ctrl + / : 현재 줄 가장 앞에 주석 '//' 달기
@. ctrl + shift + / : 현재 줄 가장 앞에 달린 주석 '//' 없애기

@. alt + D : 디렉토리 창에 현재 문서의 디렉토리 표시
@. alt + O : 윈도우 탐색기 열기 (디렉토리 창에 커서 놓은 상태로 실행해야 함)

@. alt + shift + I : 다수파일 인코딩 변환
@. ctrl + Down : 선택한 줄을 아래로 이동
@. ctrl + Up : 선택한 줄을 위로 이동
